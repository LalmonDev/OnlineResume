# OnlineResume
## 简介
个人毕设题目，基于[LaTex](https://baike.baidu.com/item/LaTeX/1212106?fr=aladdin)的在线个人简历编辑系统，将会采用Spring Boot + Vue
.js进行项目的开发，系统功能如下：<br>
* 在线创建和编辑简历<br>
* 切换简历模板<br>
* 在线预览简历<br>
* 导出PDF格式简历文件<br>
